Canonical JavaScript Stacks

  • Dependency Management - [Core] http://requirejs.org/
  • Disabling double tap on mobile - Modify the viewport, wrap in feature detection - [Polyfill] Fastclick
  • Form Validation - None. Use HTML5 form validation.
  • [if] jQuery - https://github.com/wilsonpage/jquery-fastdom
  • Logging - [Core] https://github.com/pimterry/loglevel - [Core] https://github.com/miguelmota/Navigator.sendBeacon

  • Carousel - Don't use a carousel - http://kenwheeler.github.io/slick/
  • Click to copy https://zenorocha.github.io/clipboard.js/

Developing a plugin

  • Use events rather than method handles to provide methods to "hook into" the plugin
  • Allow default configurations to be overwritten with the data-{whatever} flags
  • Make compatible as AMD or Browserify module (the jQuery mousewheel has a good example for this)
